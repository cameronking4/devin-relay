Relay v0.4.0 expands the webhook mock demo with four new services and richer event coverage across existing ones.

### New Services

- **GitHub** — Push, pull request opened/closed, issue opened, and workflow run completed events
- **Stripe** — Payment intent succeeded/failed, subscription created/deleted, invoice payment failed, and checkout session completed events
- **Supabase** — Database insert/update/delete webhook events and edge function invocation events
- **Linear** — Issue created/updated, comment created, and cycle started/completed events

### Improvements

- **Service labels** — Human-readable service names displayed in the UI
- **Datadog** — New alert types including log anomaly and APM latency alerts
- **Grafana** — Multi-alert scenarios with expanded alert details
- **PagerDuty** — Additional incident event types
- **Prometheus** — Multi-alert payload handling
- **Sentry** — New error and performance alert conditions
- **Vercel** — Broader deployment and domain event coverage
