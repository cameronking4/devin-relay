Relay v0.4.0 delivers a richer project dashboard, deeper Devin visibility, and broader webhook testing coverage.

### Features

- **Project overview stats** — Aggregate execution counts, success rates, and activity charts on the project page
- **Project cards** — Redesigned cards with at-a-glance metrics for each project
- **Header actions** — Quick-access actions added to the project header (edit, sessions, settings)
- **Edit project dialog** — Edit project details inline without leaving the dashboard
- **Devin session links** — Direct links to Devin sessions from execution detail views and summaries
- **Prompt configuration** — Enhanced prompt editor and preview components for trigger templates

### Webhook Improvements

- **POST body processing** — Improved parsing, validation, and normalization of incoming webhook payloads
- **Mock demo expansion** — Added GitHub, Stripe, Supabase, and Linear mock services with human-readable service labels
- **New event types** — Broader event coverage for Datadog, Grafana, PagerDuty, Prometheus, Sentry, and Vercel mocks
